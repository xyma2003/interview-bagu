# 面试八股题库

> AI Agent / Frontend / Backend 三方向全覆盖面试题，统一格式：题目 → 解析 → 讲解 → 考察点 → 面试官期望 → 示例答案

## 分支说明

| 分支 | 内容 |
|------|------|
| `ai-agent` | AI Agent 开发八股（LLM/RAG/多智能体/Framework 等） |
| `frontend` | 前端八股（JS/CSS/React/Vue/性能/网络等） |
| `backend` | 后端八股（数据库/Redis/消息队列/系统设计/Python/Java等） |
| `algorithm` | 算法题（解析/考点/示例答案等） |


## 题目格式

每道题包含六要素：
- **题目** — 面试中实际问法
- **题目解析** — 背景和出题动机
- **题目讲解** — 深度知识点剖析
- **考察点** — 2-4个核心考察维度
- **面试官更想听** — 拉分的回答角度
- **示例答案** — 可直接复用的完整答案
